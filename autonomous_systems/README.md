# Autonomous Systems Resources

## 1. GitHub Repositories

- Apollo (Autonomous Driving Platform, Baidu) – https://github.com/ApolloAuto/apollo  
- Autoware (Open-source software for autonomous vehicles) – https://github.com/Autoware-AI/autoware.ai
